ABOUT

Halo! Saya Auriel Diya Ulhaque Muntaz Waris, seorang mahasiswi di Universitas Palangka Raya. Saya saat ini sedang aktif mengikuti kegiatan pembelajaran studi independen bersama Infinite Learning. Dalam perjalanan studi saya, saya sedang menjalani eksplorasi dan pemahaman yang mendalam tentang Web Development. Studi independen ini memberi saya kebebasan untuk mengejar minat pribadi saya dan memperluas wawasan saya dalam bidang ini.
